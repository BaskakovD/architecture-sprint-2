# Задание 2. Шардирование

Директория mongo-sharding

`cd mongo-sharding`

## Запуск проекта

1. Запускаем `docker compose`

   `docker compose up -d`

2. Инициируем базу данных

   `./mongo-sharding.sh`

3. Проверяем количество документов

   `./mongo-db-checker.sh`

Графическое решение см. в файле task1-task6.drawio лист Task2
